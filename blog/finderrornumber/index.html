FindErrorNumber

2018-07-12 :: lambda@lambda.tw
#Golang 

找出錯誤的數字，使用 Golang⌗

題目⌗

在輸入一連串的數字中(從一開始連續[1, 2, 3, 4, 5, 6])找到錯誤(重複)的數字，並且把錯誤的先列出來再將正確的數字附加到到後面

輸入⌗

[1,2,2,4]
[1,2,2,4,5,5,7]

輸出⌗

[2,3]
[2,5,3,6]

實做⌗

main.go⌗

package main

import "fmt"

func findErrorNums(nums []int) (ret []int) {
	dict := make(map[int]int)
	for _, num := range nums {
		_, ok := dict[num]
		if ok {
			ret = append(ret, num)
		} else {
			dict[num] = num
		}
	}
	for i := 1; i <= len(nums); i++ {
		_, ok := dict[i]
		if !ok {
			ret = append(ret, i)
		}
	}
	return

}

func main() {
	input := []int{1, 2, 2, 4}
	output := findErrorNums(input)
	fmt.Println(output)
}

細解⌗

Package⌗

宣告這檔案在哪個 package 裡面，若是在別的 package 裡面你可以在別的 package import 後直接呼叫

  • $GOPATH/src/my/m.go

    package my

import "fmt"

func Pm(str string) {
    fmt.Println("Pm: ", str)
}

  • $GOPATH/src/my/y.go

    package my

import "fmt"

func Py(str string) {
    fmt.Println("Py: ", str)
}

  • $GOPATH/src/hello/main.go

    package main

import "my"

func main() {
    my.Pm("Hello")
    my.Py("Hello")
}

cd $GOPATH/src/hello/
go run main.go
Pm:  Hello
Py:  Hello
